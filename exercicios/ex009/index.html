Trabalhando com Listas

Listas Ordenadas

Usamos a tag <ol> </ol> (ordered list) para criar uma lista onde a ordem dos itens é algo importante. E cada item da lista é colocado na tag <li> </li> (list item). No HTML5 o fechamento da tag lt é opcional

Dentro da tag<ol> </ol>, podemos usar o parâmetro type=" " para indicar o tipo do marcador: 1- para valor padrão, A- para lista alfabética Maiúscula, a- para lista alfabética minúscula, I-para algrismos romanos Maiúsculo, i- para algarismos romanos minúsculos;

Listas não Ordenadas

Usamos a tag <ul> <ul> (unordered list) para criar listas onde a ordem dos itens não é importante

Dentro da tag<ul> </ul>, podemos usar o parâmetro type=" " para indicar o tipo do marcador: 

    disc - padrão (bolinha preta),
    circle - bolinha vazada
    square - quadradinho preto

Minhas linguagens favoritas

Para criar uma sub lista em algum item, usamos uma nova tag<ol> </ol> logo abaixo do item. E para personalizar o tipo de marcador, usamos o type=" ". Para dar sequência, usamos o start=" " . O start é paramêtro numérico, então se quiser continuar a partir do "e" usa o start="5"

    Lisa de Definições

    Uma lista de definição fica dentro da tag <dl> </dl> (definition list). Cada termo da lista fica dentro da tag <dt> </dt> (definition term). E cada descrição do termo fica dentro da tag <dd> </dd> (definition description). Ex:

    HTML
    Linguagem de marcação do conteúdo de um site.
    CSS
    Linguagem de marcação para a criação do design de um site.
    JavaScript
    Linguagem de programação para a criação de interatividade de um site
